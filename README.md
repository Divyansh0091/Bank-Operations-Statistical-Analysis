# Bank Back-Office Operations: Advanced Two-Way ANOVA Performance Analytics

## 🎯 Project Executive Summary
This project builds a robust **Statistical Evaluation Engine** to analyze the dual impact of **Operational Shifts** (Morning vs. Night) and **Training Status** (Certified vs. Non-Certified) on employee performance scores.

## 📈 Technical Implementation
* **Statistical Method:** Two-Way ANOVA with Interaction Effects.
* **Libraries:** `pandas` for data structuring, `statsmodels` for OLS regression, and `Plotly` for interactive dashboarding.
* **Objective:** To determine if shift timing and certification significantly influence productivity.

## 📊 Key Insights & Dashboarding
* The project generates an **Interactive Dashboard (`MNC_ANOVA_Interactive_Dashboard.html`)** that visualizes performance trends across different shifts.
* Automated predictive logic identifies the statistical significance of training impact.

## 🛠 Project Structure
* `ANOVA_Performance_Insights.ipynb`: Main analysis notebook containing data ingestion, ANOVA statistical reporting, and Plotly visualization logic.
